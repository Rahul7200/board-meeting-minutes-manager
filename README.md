# Tool-99 — Board Meeting Minutes Manager

An AI-powered web application for managing board meeting minutes,
built as an internship capstone project by a 5-member team.

---

## Team

| Role | Responsibilities |
|------|-----------------|
| Java Developer 1 | Spring Boot, JWT Auth, Redis, Docker, Email |
| Java Developer 2 | React Frontend, Flyway Migrations, UI/UX |
| AI Developer 1 | Flask AI Service, /describe, /recommend endpoints |
| AI Developer 2 | Groq Integration, /generate-report, Security |
| Security Reviewer | OWASP ZAP, SECURITY.md, Threat Model |

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Java 17, Spring Boot 3.x |
| Database | PostgreSQL 15 |
| Cache | Redis 7 |
| Migrations | Flyway |
| Security | Spring Security + JWT |
| AI Service | Python 3.11, Flask 3.x |
| AI Model | Groq API — LLaMA-3.3-70b |
| Frontend | React 18 + Vite |
| Styling | Tailwind CSS |
| Charts | Recharts |
| HTTP Client | Axios |
| Infrastructure | Docker + Docker Compose |
